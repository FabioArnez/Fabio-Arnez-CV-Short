# Fabio Arnez CV - Short Version

This CV is rendered using [RenderCV](https://github.com/rendercv/rendercv).

## Installation

To install `rendercv`, use the following command:

```bash
pip install "rendercv[full]"`
```

## Usage

To render the CV use the follwing command:

```bash
rendercv render "Fabio_Arnez_CV_short.yaml"
```
